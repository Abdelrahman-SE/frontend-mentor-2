# Blog Preview Card

A simple responsive **Blog Preview Card** built using **HTML5** and **CSS3** as part of a Frontend Mentor challenge.

## Overview

This project focuses on building a clean and responsive card component that displays a blog preview, including:

- Category tag
- Publish date
- Blog title
- Short description
- Author information

The goal of this challenge is to practice **layout**, **typography**, and **responsive design** using pure HTML and CSS.

## Built With

- HTML5 (Semantic Markup)
- CSS3
- Flexbox
- CSS Variables
- Google Fonts (Figtree)
- Responsive units (`clamp`, `rem`)

## Responsive Design

- The card is fully responsive
- Uses `max-width` with flexible margins
- Font sizes adapt using `clamp()`
- Centered vertically and horizontally using Flexbox

## What I Practiced

- Structuring a UI component
- Using CSS custom properties (variables)
- Responsive typography
- Clean and readable CSS
- Component-based thinking

## Lessons Learned

- How to build a reusable UI card

- Better control over spacing using gap

- Responsive text without media queries
